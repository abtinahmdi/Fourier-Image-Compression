# Image Compression using Fourier Transform

## Overview

This project implements an image compression algorithm based on the 2D Fourier Transform.

The main idea is to remove low-energy frequency coefficients and reconstruct the image using the remaining spectral information.

## Features

* 2D FFT implementation
* Frequency coefficient thresholding
* Image reconstruction
* Compression ratio evaluation

## Tools

* MATLAB

## Results

The compressed image preserves most visual information while significantly reducing frequency-domain data.
